# 非线性数据结构理论

## 2.2.3 非线性数据结构

### 概念定义

非线性数据结构是数据元素之间存在一对多或多对多关系的数据组织方式，具有以下特征：

1. **层次关系**：元素之间存在父子、祖先后代等层次关系
2. **网络关系**：元素之间可以形成复杂的网络连接
3. **递归结构**：数据结构本身具有递归性质

### 数学形式化

#### 图论基础

设 $G = (V, E)$ 为图，其中：
- $V$ 为顶点集合，$|V| = n$
- $E$ 为边集合，$|E| = m$
- 对于无向图：$E \subseteq \{(u,v) | u,v \in V, u \neq v\}$
- 对于有向图：$E \subseteq V \times V$

#### 树结构定义

树是连通无环图，满足：
1. **连通性**：任意两个顶点之间存在唯一路径
2. **无环性**：图中不存在环
3. **根节点**：存在一个特殊的根节点
4. **层次性**：每个节点（除根节点外）都有唯一的父节点

### 核心非线性数据结构

#### 1. 二叉树 (Binary Tree)

**数学定义**：
二叉树是每个节点最多有两个子节点的树结构：

$$T = (V, E, root)$$

其中每个节点 $v \in V$ 满足：
- $left(v) \in V \cup \{null\}$
- $right(v) \in V \cup \{null\}$
- $parent(v) \in V \cup \{null\}$

**时间复杂度**：
- 访问：$O(h)$，$h$ 为树高
- 插入/删除：$O(h)$
- 搜索：$O(h)$

```python
class TreeNode:
    """二叉树节点"""
    def __init__(self, val=0):
        self.val = val
        self.left = None
        self.right = None

class BinaryTree:
    """二叉树实现"""
    
    def __init__(self):
        self.root = None
    
    def insert(self, val):
        """插入节点"""
        if not self.root:
            self.root = TreeNode(val)
            return
        
        queue = [self.root]
        while queue:
            node = queue.pop(0)
            
            if not node.left:
                node.left = TreeNode(val)
                return
            if not node.right:
                node.right = TreeNode(val)
                return
            
            queue.append(node.left)
            queue.append(node.right)
    
    def inorder_traversal(self, node=None):
        """中序遍历 O(n)"""
        if node is None:
            node = self.root
        
        result = []
        if node:
            result.extend(self.inorder_traversal(node.left))
            result.append(node.val)
            result.extend(self.inorder_traversal(node.right))
        return result
    
    def preorder_traversal(self, node=None):
        """前序遍历 O(n)"""
        if node is None:
            node = self.root
        
        result = []
        if node:
            result.append(node.val)
            result.extend(self.preorder_traversal(node.left))
            result.extend(self.preorder_traversal(node.right))
        return result
    
    def postorder_traversal(self, node=None):
        """后序遍历 O(n)"""
        if node is None:
            node = self.root
        
        result = []
        if node:
            result.extend(self.postorder_traversal(node.left))
            result.extend(self.postorder_traversal(node.right))
            result.append(node.val)
        return result
    
    def level_order_traversal(self):
        """层序遍历 O(n)"""
        if not self.root:
            return []
        
        result = []
        queue = [self.root]
        
        while queue:
            level = []
            level_size = len(queue)
            
            for _ in range(level_size):
                node = queue.pop(0)
                level.append(node.val)
                
                if node.left:
                    queue.append(node.left)
                if node.right:
                    queue.append(node.right)
            
            result.append(level)
        
        return result
    
    def height(self, node=None):
        """计算树高 O(n)"""
        if node is None:
            node = self.root
        
        if not node:
            return 0
        
        return 1 + max(self.height(node.left), self.height(node.right))
    
    def size(self, node=None):
        """计算节点数 O(n)"""
        if node is None:
            node = self.root
        
        if not node:
            return 0
        
        return 1 + self.size(node.left) + self.size(node.right)
```

#### 2. 二叉搜索树 (Binary Search Tree)

**数学定义**：
二叉搜索树是满足以下性质的二叉树：

对于任意节点 $v$：
- $left(v)$ 子树中的所有节点值 $< v.val$
- $right(v)$ 子树中的所有节点值 $> v.val$

**时间复杂度**：
- 平均情况：$O(\log n)$
- 最坏情况：$O(n)$（退化为链表）

```python
class BinarySearchTree:
    """二叉搜索树实现"""
    
    def __init__(self):
        self.root = None
    
    def insert(self, val):
        """插入节点 O(h)"""
        self.root = self._insert_recursive(self.root, val)
    
    def _insert_recursive(self, node, val):
        if not node:
            return TreeNode(val)
        
        if val < node.val:
            node.left = self._insert_recursive(node.left, val)
        elif val > node.val:
            node.right = self._insert_recursive(node.right, val)
        
        return node
    
    def search(self, val):
        """搜索节点 O(h)"""
        return self._search_recursive(self.root, val)
    
    def _search_recursive(self, node, val):
        if not node or node.val == val:
            return node
        
        if val < node.val:
            return self._search_recursive(node.left, val)
        else:
            return self._search_recursive(node.right, val)
    
    def delete(self, val):
        """删除节点 O(h)"""
        self.root = self._delete_recursive(self.root, val)
    
    def _delete_recursive(self, node, val):
        if not node:
            return node
        
        if val < node.val:
            node.left = self._delete_recursive(node.left, val)
        elif val > node.val:
            node.right = self._delete_recursive(node.right, val)
        else:
            # 找到要删除的节点
            if not node.left:
                return node.right
            elif not node.right:
                return node.left
            else:
                # 有两个子节点，找到右子树的最小值
                successor = self._find_min(node.right)
                node.val = successor.val
                node.right = self._delete_recursive(node.right, successor.val)
        
        return node
    
    def _find_min(self, node):
        """找到最小值节点"""
        while node.left:
            node = node.left
        return node
    
    def inorder_traversal(self):
        """中序遍历（有序输出）"""
        result = []
        self._inorder_recursive(self.root, result)
        return result
    
    def _inorder_recursive(self, node, result):
        if node:
            self._inorder_recursive(node.left, result)
            result.append(node.val)
            self._inorder_recursive(node.right, result)
```

#### 3. 堆 (Heap)

**数学定义**：
堆是完全二叉树，满足堆性质：

对于最大堆：
$$\forall v \in V: v.val \geq left(v).val \land v.val \geq right(v).val$$

对于最小堆：
$$\forall v \in V: v.val \leq left(v).val \land v.val \leq right(v).val$$

**时间复杂度**：
- 插入：$O(\log n)$
- 删除：$O(\log n)$
- 查找最大/最小值：$O(1)$

```python
class MaxHeap:
    """最大堆实现"""
    
    def __init__(self):
        self.heap = []
    
    def parent(self, index):
        """获取父节点索引"""
        return (index - 1) // 2
    
    def left_child(self, index):
        """获取左子节点索引"""
        return 2 * index + 1
    
    def right_child(self, index):
        """获取右子节点索引"""
        return 2 * index + 2
    
    def has_parent(self, index):
        """是否有父节点"""
        return self.parent(index) >= 0
    
    def has_left_child(self, index):
        """是否有左子节点"""
        return self.left_child(index) < len(self.heap)
    
    def has_right_child(self, index):
        """是否有右子节点"""
        return self.right_child(index) < len(self.heap)
    
    def swap(self, index1, index2):
        """交换两个元素"""
        self.heap[index1], self.heap[index2] = self.heap[index2], self.heap[index1]
    
    def peek(self):
        """查看最大值 O(1)"""
        if not self.heap:
            raise IndexError("堆为空")
        return self.heap[0]
    
    def push(self, val):
        """插入元素 O(log n)"""
        self.heap.append(val)
        self._heapify_up()
    
    def _heapify_up(self):
        """向上调整堆"""
        index = len(self.heap) - 1
        while self.has_parent(index) and self.heap[self.parent(index)] < self.heap[index]:
            self.swap(index, self.parent(index))
            index = self.parent(index)
    
    def pop(self):
        """删除最大值 O(log n)"""
        if not self.heap:
            raise IndexError("堆为空")
        
        if len(self.heap) == 1:
            return self.heap.pop()
        
        max_val = self.heap[0]
        self.heap[0] = self.heap.pop()
        self._heapify_down()
        
        return max_val
    
    def _heapify_down(self):
        """向下调整堆"""
        index = 0
        while self.has_left_child(index):
            larger_child_index = self.left_child(index)
            
            if (self.has_right_child(index) and 
                self.heap[self.right_child(index)] > self.heap[self.left_child(index)]):
                larger_child_index = self.right_child(index)
            
            if self.heap[index] > self.heap[larger_child_index]:
                break
            
            self.swap(index, larger_child_index)
            index = larger_child_index
    
    def size(self):
        """堆大小"""
        return len(self.heap)
    
    def is_empty(self):
        """是否为空"""
        return len(self.heap) == 0

# 堆排序
def heap_sort(arr):
    """堆排序 O(n log n)"""
    heap = MaxHeap()
    
    # 构建堆
    for val in arr:
        heap.push(val)
    
    # 依次取出最大值
    result = []
    while not heap.is_empty():
        result.append(heap.pop())
    
    return result[::-1]  # 反转得到升序
```

#### 4. 图 (Graph)

**数学定义**：
图 $G = (V, E)$ 由顶点集合和边集合组成：

- **邻接矩阵表示**：$A[i][j] = 1$ 如果 $(i,j) \in E$，否则 $A[i][j] = 0$
- **邻接表表示**：每个顶点维护一个邻居列表

**时间复杂度**：
- 邻接矩阵：访问边 $O(1)$，遍历邻居 $O(|V|)$
- 邻接表：访问边 $O(degree(v))$，遍历邻居 $O(degree(v))$

```python
from collections import defaultdict

class Graph:
    """图实现（邻接表）"""
    
    def __init__(self, directed=False):
        self.graph = defaultdict(list)
        self.directed = directed
        self.vertices = set()
    
    def add_edge(self, u, v, weight=1):
        """添加边"""
        self.graph[u].append((v, weight))
        self.vertices.add(u)
        self.vertices.add(v)
        
        if not self.directed:
            self.graph[v].append((u, weight))
    
    def remove_edge(self, u, v):
        """删除边"""
        if u in self.graph:
            self.graph[u] = [(neighbor, weight) for neighbor, weight in self.graph[u] if neighbor != v]
        
        if not self.directed and v in self.graph:
            self.graph[v] = [(neighbor, weight) for neighbor, weight in self.graph[v] if neighbor != u]
    
    def get_neighbors(self, vertex):
        """获取邻居"""
        return self.graph[vertex]
    
    def has_edge(self, u, v):
        """检查是否存在边"""
        return any(neighbor == v for neighbor, _ in self.graph[u])
    
    def get_vertices(self):
        """获取所有顶点"""
        return list(self.vertices)
    
    def get_edges(self):
        """获取所有边"""
        edges = []
        for u in self.graph:
            for v, weight in self.graph[u]:
                if not self.directed or (u, v) not in [(edge[1], edge[0]) for edge in edges]:
                    edges.append((u, v, weight))
        return edges
    
    def dfs(self, start, visited=None):
        """深度优先搜索 O(V + E)"""
        if visited is None:
            visited = set()
        
        visited.add(start)
        result = [start]
        
        for neighbor, _ in self.graph[start]:
            if neighbor not in visited:
                result.extend(self.dfs(neighbor, visited))
        
        return result
    
    def bfs(self, start):
        """广度优先搜索 O(V + E)"""
        visited = set()
        queue = [start]
        visited.add(start)
        result = []
        
        while queue:
            vertex = queue.pop(0)
            result.append(vertex)
            
            for neighbor, _ in self.graph[vertex]:
                if neighbor not in visited:
                    visited.add(neighbor)
                    queue.append(neighbor)
        
        return result
    
    def topological_sort(self):
        """拓扑排序（仅适用于有向无环图）"""
        if not self.directed:
            raise ValueError("拓扑排序仅适用于有向图")
        
        in_degree = defaultdict(int)
        for u in self.graph:
            for v, _ in self.graph[u]:
                in_degree[v] += 1
        
        queue = [u for u in self.vertices if in_degree[u] == 0]
        result = []
        
        while queue:
            u = queue.pop(0)
            result.append(u)
            
            for v, _ in self.graph[u]:
                in_degree[v] -= 1
                if in_degree[v] == 0:
                    queue.append(v)
        
        if len(result) != len(self.vertices):
            raise ValueError("图中存在环")
        
        return result

# 最短路径算法
def dijkstra(graph, start):
    """Dijkstra最短路径算法 O(V^2)"""
    distances = {vertex: float('infinity') for vertex in graph.get_vertices()}
    distances[start] = 0
    unvisited = set(graph.get_vertices())
    
    while unvisited:
        # 找到距离最小的未访问顶点
        current = min(unvisited, key=lambda v: distances[v])
        unvisited.remove(current)
        
        # 更新邻居距离
        for neighbor, weight in graph.get_neighbors(current):
            if neighbor in unvisited:
                new_distance = distances[current] + weight
                if new_distance < distances[neighbor]:
                    distances[neighbor] = new_distance
    
    return distances
```

### 性能比较

| 操作 | 二叉树 | 二叉搜索树 | 堆 | 图 |
|------|--------|------------|----|----|
| 访问 | O(h) | O(h) | O(1) | O(1) |
| 插入 | O(h) | O(h) | O(log n) | O(1) |
| 删除 | O(h) | O(h) | O(log n) | O(degree) |
| 搜索 | O(h) | O(h) | O(n) | O(V+E) |

### 应用场景

1. **二叉树**：表达式树、语法分析树
2. **二叉搜索树**：有序数据存储、字典实现
3. **堆**：优先队列、堆排序、TopK问题
4. **图**：社交网络、路由算法、依赖关系

### 理论总结

非线性数据结构为复杂问题的建模和解决提供了强大的工具。理解这些数据结构的数学基础和实现原理，对于设计高效算法和系统架构至关重要。
