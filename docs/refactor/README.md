# Python 形式化计算科学文档体系

## 概述

本目录包含基于 `/docs/model` 目录内容重构的 Python 形式化计算科学文档体系。该体系遵循严格的学术规范，从理念基础到具体实践，构建了完整的形式化论证体系。

## 目录结构

```
docs/refactor/
├── 00-理念基础/           # 哲学理念与认知基础
├── 01-形式科学/           # 数学、逻辑、形式化方法
├── 02-理论基础/           # 计算理论、算法理论
├── 03-具体科学/           # 软件工程、系统架构
├── 04-行业领域/           # 各行业应用领域

```
