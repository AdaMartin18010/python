# Web 框架生态

**Python Web开发框架完整指南**

---

## 📋 目录

### 核心框架

1. **[FastAPI](01-fastapi.md)** ⭐ 推荐
   - 现代、快速的Web框架
   - 基于标准Python类型提示
   - 自动API文档生成
   - 异步支持

2. **[Django](02-django.md)**
   - 全功能Web框架
   - 包含ORM、Admin、Auth
   - 适合大型项目

3. **[Flask](03-flask.md)**
   - 轻量级微框架
   - 灵活可扩展
   - 适合中小型项目

### 异步框架

4. **[Starlette](04-starlette.md)**
   - 轻量级ASGI框架
   - FastAPI的基础

5. **[aiohttp](05-aiohttp.md)**
   - 异步HTTP客户端/服务器

6. **[httpx](06-httpx.md)**
   - 现代HTTP客户端
   - 支持HTTP/2

---

## 🚀 快速选择

### 选择建议

| 项目类型 | 推荐框架 | 理由 |
|---------|---------|------|
| 现代API | FastAPI | 性能好、类型安全、文档自动 |
| 大型应用 | Django | 功能完整、生态成熟 |
| 微服务 | FastAPI/Flask | 轻量、灵活 |
| 实时应用 | Starlette | 异步、WebSocket |
