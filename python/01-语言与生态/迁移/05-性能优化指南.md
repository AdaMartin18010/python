## 2. 热点优化策略

- 算法与数据结构优先
- 减少对象创建与属性查找
- I/O 聚合：批量读写、异步 I/O

## 3. 并发与并行

```python
import asyncio
from asyncio import Semaphore

async def worker(sem: Semaphore, task_id: int):
    async with sem:
        await do_io(task_id)

async def main():
    sem = Semaphore(10)
    await asyncio.gather(*(worker(sem, i) for i in range(100)))
```
