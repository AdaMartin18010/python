## 1. uv 工具概述

- 统一：安装、解析、锁定、构建、发布，一体化工具链
- 目标：更快、更可复现，替代 pip/virtualenv 的零散体验

### 快速开始（CLI）

```bash
# 安装（任选）
pipx install uv  # 推荐
# 或
pip install uv

# 创建并安装依赖（基于 pyproject）
uv pip install -e .[dev]

# 生成/更新锁文件
uv pip compile pyproject.toml -o uv.lock

# 基于锁同步环境
uv pip sync uv.lock
```
